GARGANTUA'S GLITCH

Event Description

Enter the cosmic realm of coding where every bug is a gravitational trap! In this high-pressure event, participants must debug tricky code and solve challenging problems before being pulled into chaos. From small syntax errors to major logical glitches, only those with sharp focus and quick thinking can escape Gargantua’s pull and claim victory.

Round 1 - Event Horizon Debugging

Participants will be given 10 code snippets in C and Python containing logical or syntax errors. Identify and correct the errors within the given time.Speed and accuracy in debugging will be checked

Round 2 - Planetary Build-out

Shortlisted participants will receive 5 coding problems through a draw. Write, compile, and execute code to produce the correct output.
